# Scenario 1: Agent with Classic Answer

## What It Is

Classic answers are predefined, static responses manually authored by agent makers. The agent returns exactly what was written — no AI generation is involved. These are the cheapest type of interaction and are best suited for situations where precision and consistency are required.

## When It's Used

- FAQ bots with fixed, curated answers (e.g., "What is the return policy?")
- Scripted menu-driven conversations with button prompts
- Welcome/greeting messages when a user first opens the agent
- Compliance-sensitive responses where wording must not vary
- Simple routing messages that direct users to the right department

**1 Copilot Credit per classic answer**

## Cost Comparison: PAYG vs. Capacity Pack

**Example:** An HR agent serves 900 employees/day, each asking 4 classic-answer questions = 3,600 credits/day = 108,000 credits/month.

| Billing Model | Calculation | Monthly Cost | Annual Cost |
|---------------|-------------|--------------|-------------|
| Pay-As-You-Go | 108,000 × $0.01 | $1,080 | $12,960 |
| Capacity Packs | 5 packs × $200 (125,000 credits, 17,000 buffer) | $1,000 | $12,000 |
| Capacity Packs + PAYG | 4 packs ($800) + 8,000 overage × $0.01 ($80) | $880 | $10,560 |

### Additional Examples

**Example A — Small Team FAQ Bot:** A 50-person team uses a policy FAQ agent with 5 classic answers per person per day = 250 credits/day = ~7,500 credits/month. One capacity pack (25,000) more than covers this at $200/month vs. $75 PAYG. **PAYG is cheaper here.**

**Example B — Call Center Script Agent:** A scripted call center bot serves 5,000 calls/day with 6 classic answers per call = 30,000 credits/day = ~900,000 credits/month. This requires 36 capacity packs at $7,200/month vs. $9,000 PAYG — **saving $1,800/month.**
