# Scenario 10: Content Processing Tools

## What It Is

Content processing tools handle document processing tasks: custom document processing models, invoice processing, receipt processing, identity document reading, business card reading, and object detection. These are available only in agent flows.

## When It's Used

- Extracting data from PDF invoices, receipts, or contracts
- Processing identity documents (passports, driver's licenses)
- Reading business cards and populating CRM records
- Any document scanning/OCR within an agent flow

## Credit Consumption

**8 Copilot Credits per page processed**

## Licensed vs. Unlicensed User Comparison

| User Type | Credits Per Page Processed | Monthly Cost (5,000 pages) |
|-----------|---------------------------|--------------------------|
| M365 Copilot Licensed User (B2E, internal channel) | 0 (included) | $0 |
| Unlicensed User | 8 credits | $400 |
| External / Autonomous | 8 credits | $400 |

## Cost Comparison: PAYG vs. Capacity Pack

**Example — Contract Processing Agent:** Processes 500 contracts/month, each averaging 5 pages.

- Credits = 500 × 5 × 8 = 20,000 credits/month

| Billing Model | Calculation | Monthly Cost | Annual Cost |
|---------------|-------------|--------------|-------------|
| Pay-As-You-Go | 20,000 × $0.01 | $200 | $2,400 |
| 1 Capacity Pack | $200 (5,000 credits buffer) | $200 | $2,400 |

### Additional Examples

**Example A — Receipt Processing:** 2,000 single-page receipts/month = 16,000 credits = $160 PAYG.

**Example B — Large Document Processing:** 100 documents at 20 pages each = 16,000 credits = $160 PAYG. A 50-page legal brief = 400 credits ($4.00) to process.
